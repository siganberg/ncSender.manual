# 4th Axis (A-Axis)

!!! info "Pro Feature"
    4th Axis support is available in **ncSender Pro** only.

Full 4th axis (A-axis) control with 3D visualization for rotary and indexing operations.

## Features

- **DRO display** — A-axis coordinate shown alongside X, Y, Z
- **Jog control** — A+/A- buttons with configurable step sizes
- **3D visualization** — Rotary axis rendering in the visualizer
- **G-code support** — Full support for A-axis commands in loaded programs

## Jog Behavior

- A-axis jogs at 25% of the XY feed rate by default
- Step sizes are in degrees
- Long-press for continuous rotation
- Double-click the A-axis DRO card to enter a specific angle
