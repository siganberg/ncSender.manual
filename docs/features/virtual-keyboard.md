# Virtual Keyboard

!!! info "Pro Feature"
    Virtual Keyboard is available in **ncSender Pro** only.

Built-in virtual keyboard for kiosk and touchscreen setups where no physical keyboard is attached.

## Features

- **QWERTY layout** — Full keyboard with shift support
- **Numpad layout** — Quick number entry for coordinates and settings
- **Auto-show** — Appears when any text input or code editor receives focus
- **Touch-friendly** — Sized for touchscreen interaction

## Supported Inputs

The virtual keyboard works with:

- Console command input
- Coordinate entry (DRO double-click)
- Settings inputs
- Macro editor (Monaco code editor)
- Plugin dialog inputs
- Program event editors

## Settings

Toggle the virtual keyboard on/off in **Settings > Virtual Keyboard**.

When disabled, the keyboard won't appear on focus — useful when a physical keyboard is connected.
