# Replicator

Replicate your loaded G-code program in a grid pattern for batch production.

## Features

- Configure rows and columns for the replication grid
- Set spacing between copies
- Automatic offset calculation
- Preview the replicated layout before generating

## Usage

1. Load your G-code program
2. Open **Replicator** from the Tools menu
3. Set the number of rows and columns
4. Configure X and Y spacing between copies
5. Click **Generate** to create the replicated program
