# BoxJoints

G-code generator for box joints (finger joints) for CNC woodworking.

## Features

- Configure joint dimensions (finger width, material thickness)
- Preview joint layout before cutting
- Generate matching male and female joint patterns
- Support for metric and imperial units

## Usage

1. Open **BoxJoints** from the Tools menu
2. Set material thickness and finger width
3. Configure bit diameter and cutting parameters
4. Preview the joint pattern
5. Click **Generate** to create the G-code
